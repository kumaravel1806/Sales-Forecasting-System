## 🚀 HOW TO USE

### Accessing the Dashboard:
1. Open: **http://localhost:8000/admin_dashboard.html**
2. You'll see the sidebar on the left
3. Main dashboard with live data on the right
4. Click any sidebar item to navigate

### Navigation:
- **Click Dashboard** - Return to main overview
- **Click Products** - Manage your products
- **Click Upload Data** - Import CSV files
- **Click Expiry Management** - Track expiring items
- **Click Feedback** - View customer reviews
- **Click Real-Time** - See live analytics
- **Click Scenario Analysis** - Run forecasts
- **Click Logout** - Secure sign out

### Features:
- **Hover** over sidebar items for animation
- **Click Refresh** button to reload data
- **Auto-refresh** happens every 30 seconds
- **Scroll** main content area (sidebar stays fixed)
